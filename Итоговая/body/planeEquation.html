Уравнение плоскости

Определение

Уравнение плоскости - это математическое выражение, которое описывает положение и форму плоскости в трехмерном пространстве.

Общий вид уравнения плоскости

Общий вид уравнения плоскости в декартовой системе координат: Ax + By + Cz + D = 0, где A, B, C - коэффициенты плоскости, D - свободный член.

Уравнение плоскости через точку и нормальный вектор

Уравнение плоскости в декартовой системе координат через заданную точку (x₀, y₀, z₀) и нормальный вектор (a, b, c) можно записать в виде: a(x - x₀) + b(y - y₀) + c(z - z₀) = 0.
